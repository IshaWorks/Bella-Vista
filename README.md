# Bella Vista 🍝

**Bella Vista** is a beginner-friendly Italian restaurant website built using **HTML** and **CSS**. It features a clean and responsive design that showcases the restaurant’s menu, ambience, and contact information, providing visitors with a smooth and appetizing browsing experience.

---

## 🔹 Features
- Simple and elegant layout
- Responsive design for desktop and mobile
- Menu section to showcase dishes
- Contact information and basic footer
- Styled with CSS for a visually appealing look

---

## 🔹 Skills Demonstrated
- HTML5 structure
- CSS styling and layout
- Responsive web design
- Basic web development best practices
